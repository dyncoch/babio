# Babio — public site

Marketing landing page, privacy policy, and support page for the
[Babio](https://dyncoch.github.io/babio/) iPhone app.

This repository contains **only** the public-facing static site. The app's
source code is private.

- Landing page: `index.html`
- Privacy policy: `privacy.html`
- Support: `support.html`

Hosted via GitHub Pages from `main`.
